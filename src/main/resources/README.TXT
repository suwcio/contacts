1. Do uruchomienia aplikacji potrzebny jest serwer bazodanowy.
2. Wszelkie konfiguracje bazy (adres, login, hasło) podajemy w pliku application.properties.
3. W celu uruchomienia aplikacji w wierszu poleceń wpisujemy: mvn spring-boot:run.
4. Aplikacja uruchomi się pod adresem http://localhost:8081/contacts/
